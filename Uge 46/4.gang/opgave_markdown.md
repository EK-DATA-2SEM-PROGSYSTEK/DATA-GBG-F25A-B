# Opgave: øvelser med Markdown

Markdown er det sprog I skal skrive jeres README og CONTRIBUTING filer i. 

IntelliJ understøtter visning af Markdown-filer (de skal have endelsen .md), så I kan skrive filerne i markdown her, og samtidig få et preview af hvordan den endelige visning bliver.

Orienter jer først i syntaksen i [Markdown Guide](https://markdownguide.offshoot.io/basic-syntax/).

Arbejd jer derefter igennem [Markdown Tutorial](https://www.markdowntutorial.com)
