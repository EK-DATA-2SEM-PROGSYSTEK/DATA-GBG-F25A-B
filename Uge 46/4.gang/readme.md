#  README og CONTRIBUTING

## Beskrivelse

Vi skal lave en README.md og CONTRIBUTING.md fil til eksamensprojektet.

## Læringsmål

- At kunne forklare opbygning af Markdown sprog
- At kunne udarbejde en README.md fil med relevant information til et github-projekt
- At kunne udarbejde en CONTRIBUTING.md med relevant information til et github-projekt

## Forberedelse

Læs [Guidelines for contributing](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/setting-guidelines-for-repository-contributors)

Læs [Contributing example](https://github.com/jessesquires/.github/blob/main/CONTRIBUTING.md)
