# Opgave: øvelser med README

Start med at orientere jer på Github om [Repository README files](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-readmes).

Projekter på github har en README fil - tjek de forskellige eksempler herunder:

[Markdown Guide](https://github.com/mattcone/markdown-guide)

[IntelliJ IDEA Samples for Tips and Tricks, Features, etc.](https://github.com/JetBrains/intellij-samples) 

[Spring Framework](https://github.com/spring-projects/spring-framework)

Hvis I er sultne efter flere eksempler, så er der en hel side med dem her: [Awesome README](https://github.com/matiassingers/awesome-readme?tab=readme-ov-file)

Lav derefter jeres egen README til eksamensprojektet (I kan ikke fylde alt ind endnu, men I kan starte filen op).
